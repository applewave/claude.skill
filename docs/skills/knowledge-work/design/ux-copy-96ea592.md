## 주요 내용

- **사용 방식**: /ux-copy $ARGUMENTS
- **진행 방식**: **Clear**: Say exactly what you mean. No jargon, no ambiguity. **Concise**: Use the fewest words that convey the full meaning. **Consistent**: Same terms for the same things everywhere. **Useful**: Every word should help the user accomplish their goal. **Human**: Write like a helpful person, not a robot.
