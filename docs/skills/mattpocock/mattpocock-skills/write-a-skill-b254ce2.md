Create new agent skills with proper structure, progressive disclosure, and bundled resources. Use when user wants to create, write, or build a new skill.

## 주요 내용

- **진행 방식**: **Gather requirements** - ask user about: What task/domain does the skill cover? What specific use cases should it handle? Does it need executable scripts or just instructions? Any reference materials to include? **Draft the skill** - create: SKILL.md with concise instructions Additional reference files if content exceeds 500 lines Utility scripts if deterministic operations needed **Review with user** - present draft and ask: Does this cover your use cases? Anything missing or unclear? Should any section be more/less detailed?
